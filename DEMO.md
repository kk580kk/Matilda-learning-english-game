## 📋 5分钟演示脚本

### 第1分钟：开场介绍
- **游戏名称**：玛蒂尔达的魔法书
- **目标用户**：6-12岁少儿英语学习者
- **核心玩法**：通过学习英语打败怪兽，解锁关卡

### 第2分钟：启动游戏
1. 打开游戏首页
2. 展示魔法书主题界面
3. 点击"开始冒险"按钮

### 第3分钟：关卡选择
1. 展示10个关卡（基于《玛蒂尔达》音乐剧）
2. 每个关卡对应一首歌曲
3. 点击第一关"神奇的玛蒂尔达"

### 第4分钟：核心玩法展示
**战斗场景**：
- 玛蒂尔达 vs 怪兽
- 血量条显示

**答题环节**：
1. **选择题**：听音频片段，选择正确答案
   - 示例："玛蒂尔达说自己是'a miracle'，这个词的意思是？"
   - 选项：A. 奇迹 B. 麻烦 C. 普通孩子 D. 淘气包

2. **拼写题**：根据提示拼出单词
   - 示例："玛蒂尔达非常聪明，她喜欢r_____（阅读）书籍"
   - 拖拽字母完成拼写

3. **连线题**：英文单词与中文意思连线
   - 左：Naughty / Quiet / Clever / Magical
   - 右：淘气的 / 安静的 / 聪明的 / 有魔法的

4. **跟读题**：AI朗读句子，孩子跟读评分
   - "Even if you're little, you can do a lot"

### 第5分钟：战斗反馈
- **答对**：玛蒂尔达攻击怪兽，造成伤害
- **答错**：怪兽攻击玛蒂尔达，掉血
- **胜利**：打败怪兽，获得金币奖励
- **道具系统**：展示冻结时间、跳过问题、去错答案三种道具
