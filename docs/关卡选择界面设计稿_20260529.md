# 玛蒂尔达关卡选择界面设计稿

**创建时间**: 2026-05-29 23:02  
**项目**: 玛蒂尔达魔法书 - Matilda Magic Book  
**版本**: 1.0（修复后）

---

## 🎨 设计目标

解决用户反馈的问题：
- ❌ 关卡按钮垂直间距过大，未选中的关卡几乎看不见
- ❌ 透明度设置不合理，灰色关卡可读性差
- ❌ 整体布局不紧凑，视觉效果松散

---

## ✅ 设计方案

### 1️⃣ **布局结构**
```plaintext
┌─────────────────────────────────────────────────┐
│                 #level-select h2                │
│             关卡选择 (居中，大标题)             │
├─────────────────────────────────────────────────┤
│                                                 │
│          .levels-grid (CSS Grid 布局)          │
│  ┌─────────────┬─────────────┬─────────────┐   │
│  │ .level-card │ .level-card │ .level-card │   │
│  │   (_locked) │  (current)  │ (completed) │   │
│  └─────────────┴─────────────┴─────────────┘   │
│  ┌─────────────┬─────────────┬─────────────┐   │
│  │ .level-card │ .level-card │ .level-card │   │
│  │   (locked)  │  (locked)   │ (locked)    │   │
│  └─────────────┴─────────────┴─────────────┘   │
│                                                 │
└─────────────────────────────────────────────────┘
```

### 2️⃣ **网格布局参数**

#### **桌面端 & 平板端**
```css
.levels-grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(140px, 1fr));
    gap: 20px;
    margin-bottom: 30px;
}
```
- ✅ **_minmax(140px, 1fr)**: 宽屏时 3 列，窄屏时 2 列
- ✅ **gap: 20px**: 关卡之间水平/垂直间距 20px
- ✅ **margin-bottom: 30px**: 底部留白 30px

#### **移动端**
```css
@media (max-width: 600px) {
    .levels-container {
        grid-template-columns: 1fr;
    }
}
```
- ✅ 单列布局，确保关卡在小屏幕上也能清晰显示

---

### 3️⃣ **关卡卡片样式**

#### **通用样式**
```css
.level-card {
    background: linear-gradient(145deg, 
        rgba(108, 92, 231, 0.3), 
        rgba(253, 121, 168, 0.3));
    border-radius: 15px;
    padding: 25px 20px;          /* 上下 25px，左右 20px */
    text-align: center;
    cursor: pointer;
    transition: all 0.3s ease;
    border: 2px solid transparent;
}
```

#### **状态样式**
```css
/* 未选中（锁定）状态 */
.level-card.locked {
    opacity: 0.75;  /* 从 0.5 → 0.75，保证可读性 */
    cursor: not-allowed;
}

/* 当前可用关卡 */
.level-card.current {
    border-color: var(--warning);
    animation: pulse 2s infinite;
}

/* 已完成关卡 */
.level-card.completed {
    border-color: var(--accent);
}
```

#### **内部元素样式**
```css
.level-number {
    font-size: 2.5em;       /* 从 2em → 2.5em，更醒目 */
    font-weight: bold;
}

.level-name {
    font-size: 0.9em;       /* 从 0.8em → 0.9em，更清晰 */
    margin-top: 5px;
    font-weight: 500;
}
```

## 📄 完整 CSS 实现代码

### **关卡选择部分**
```css
/* 关卡选择 */
#level-select h2 {
    text-align: center;
    margin-bottom: 20px;
}

.levels-grid {
    display: grid;
    grid-template-columns: repeat(auto-fill, minmax(140px, 1fr));
    gap: 20px;
    margin-bottom: 30px;
}

.level-card {
    background: linear-gradient(145deg, rgba(108, 92, 231, 0.3), rgba(253, 121, 168, 0.3));
    border-radius: 15px;
    padding: 25px 20px;
    text-align: center;
    cursor: pointer;
    transition: all 0.3s ease;
    border: 2px solid transparent;
}

.level-card:hover {
    transform: scale(1.05);
    border-color: var(--primary);
}

.level-card.locked {
    opacity: 0.75;
    cursor: not-allowed;
}

.level-card.completed {
    border-color: var(--accent);
}

.level-card.current {
    border-color: var(--warning);
    animation: pulse 2s infinite;
}

@keyframes pulse {
    0%, 100% { box-shadow: 0 0 0 0 rgba(243, 156, 18, 0.4); }
    50% { box-shadow: 0 0 0 10px rgba(243, 156, 18, 0); }
}

.level-number {
    font-size: 2.5em;
    font-weight: bold;
}

.level-name {
    font-size: 0.9em;
    margin-top: 5px;
    font-weight: 500;
}
```
