## 1. 文档概述

### 1.1 目的
本文档详细描述玛蒂尔达英语学习游戏的业务需求，作为产品开发、测试和验收的基准。

### 1.2 范围
- 产品名称：玛蒂尔达的魔法书（Matilda's Magic Book）
- 产品类型：少儿英语学习游戏
- 目标平台：Web 端（桌面 + 移动浏览器）、后续扩展原生 APP

### 1.3 术语定义

| 术语 | 定义 |
|------|------|
| 玩家 | 使用本产品的 6-12 岁儿童用户 |
| 关卡 | 一个完整的学习单元，包含若干题目和战斗环节 |
| 题型 | 题目的呈现形式，包括选择题、拼写题、连线题、跟读题 |
| Boss | 关卡末尾的高难度挑战 |
| 道具 | 辅助玩家通关的游戏内物品 |

## 2. 项目背景

### 2.1 市场现状

#### 2.1.1 行业背景
- 中国少儿英语教育市场规模超过 1000 亿元，年增长率 15%+
- 6-12 岁儿童约 1.2 亿，在线教育渗透率快速提升
- 政策推动英语教育低龄化，新课标要求小学阶段词汇量 500+

#### 2.1.2 用户痛点
| 痛点 | 描述 | 严重程度 |
|------|------|----------|
| 学习枯燥 | 传统刷题、背单词方式让孩子抵触 | 高 |
| 缺乏动力 | 缺少正向反馈，难以持续投入 | 高 |
| 学用分离 | 学习内容与实际应用脱节 | 中 |
| 监督困难 | 家长难以追踪孩子学习进度 | 中 |

#### 2.1.3 竞品分析

| 产品 | 优势 | 劣势 | 我们的差异化 |
|------|------|------|-------------|
| 多邻国 | 游戏化成熟、用户基数大 | 内容通用、无 IP 吸引力 | 音乐剧 IP 深度绑定、剧情驱动 |
| 宝宝巴士 | 低龄内容成熟 | 年龄上限低（≤6 岁） | 覆盖 6-12 岁年龄段 |
| 英语趣配音 | 配音形式新颖 | 缺乏游戏反馈机制 | 战斗系统 + 即时奖励 |
| 百词斩 | 词汇学习效率高 | 缺乏情境和趣味性 | 剧情情境 + 角色代入 |

### 2.2 产品机会

#### 2.2.1 IP 价值
- 《玛蒂尔达》音乐剧全球知名，儿童认知度高
- 主角玛蒂尔达热爱阅读、勇敢正义，具有教育意义
- 音乐剧包含优美英文歌曲和对话，适合语言学习

#### 2.2.2 技术成熟
- HTML5 支持跨平台部署，开发成本降低
- Web Speech API 支持浏览器内语音识别
- AI 语音合成技术成熟，降低音频制作成本

#### 2.2.3 时机恰当
- 游戏化学习成为教育科技主流
- 家长对在线教育接受度提升
- 政策推动英语教育改革

## 3. 业务需求

### 3.1 愿景声明

> **让英语学习不再是任务，而是孩子主动想要参与的游戏体验。**
> 
> 核心理念：兴趣不是学习的奖励，兴趣本身就应该是学习系统的一部分。

### 3.2 业务目标

#### 3.2.1 商业目标

| 指标 | 目标值 | 时间周期 | 测量方式 |
|------|--------|----------|----------|
| 用户获取 | 1000 个家庭注册 | 上线后 30 天 | 后台统计 |
| 用户留存 | 7 日留存率 ≥ 60% | 持续监测 | 后台统计 |
| 用户活跃 | DAU/MAU ≥ 40% | 持续监测 | 后台统计 |
| 付费转化 | ≥ 5% | 上线后 90 天 | 后台统计 |

#### 3.2.2 教育目标

| 指标 | 目标值 | 测量方式 |
|------|--------|----------|
| 学习时长 | 日均 ≥ 15 分钟 | 后台统计 |
| 单词留存率 | 7 天后 ≥ 60% | 前后测对比 |
| 主动学习频次 | 日均 ≥ 1 次 | 后台统计 |
| 家长满意度 | ≥ 80% | 问卷调研 |

#### 3.2.3 产品目标

| 阶段 | 目标 | 交付物 | 内容量 |
|------|------|--------|--------|
| MVP | 验证核心玩法 | 网页版可运行游戏 | 50 词 + 20 句型 + 10 关卡 |
| V1.0 | 完整关卡体系 + 用户系统 | 网页版 + PWA 移动端 | 200 词 + 50 句型 + 30 关卡 |
| V1.5 | 社交功能 + 内容扩展 | 原生 APP | 500 词 + 100 句型 + 60 关卡 |
| V2.0 | 多 IP 扩展 | 平台化产品 | 1500 词 + 400 句型 + 多 IP |

### 3.3 目标用户

#### 3.3.1 用户画像

**核心用户：6-12 岁儿童**

| 年龄段 | 年级 | 词汇目标 | 句型目标 | 内容来源 |
|--------|------|----------|----------|----------|
| 6-8 岁 | 1-2 年级 | 150 词 | 30 句 | 新课标 1-2 年级 + 剑桥 Starters |
| 8-10 岁 | 3-4 年级 | 200 词 | 40 句 | 新课标 3-4 年级 + 剑桥 Movers |
| 10-12 岁 | 5-6 年级 | 150 词 | 30 句 | 新课标 5-6 年级 + 剑桥 Flyers |
| **合计** | **1-6 年级** | **500 词** | **100 句** | **全覆盖** |

**次要用户：家长**

- 年龄：30-45 岁
- 关注：孩子学习进度、学习效果、使用安全
- 需求：学习报告、进度追踪、时间管控

#### 3.3.2 用户故事

**核心场景**：
> "我女儿 8 岁，特别喜欢《玛蒂尔达》音乐剧，但非常讨厌背单词。我希望有一款她能主动想玩的游戏，让她在不知不觉中学习英语。"

**典型用户旅程**：
1. 家长打开游戏，孩子看到熟悉的玛蒂尔达形象
2. 孩子选择关卡，开始冒险
3. 通过答题击败怪物，获得成就感
4. 每次学习 15-20 分钟，主动要求再来一关
5. 家长收到学习报告，了解孩子进步

### 4.3 题型详细需求

#### 4.3.1 选择题

**需求描述**：播放音频或显示题目，玩家选择正确答案。

**业务规则**：
- 每题 4 个选项，1 个正确答案
- 答对：对怪物造成伤害
- 答错：玩家掉血，显示正确答案
- 超时（30 秒）：视为答错

**示例**：
```
题目：玛蒂尔达说自己是"a miracle"，这个词的意思是？
选项：A. 奇迹  B. 麻烦  C. 普通孩子  D. 淘气包
正确答案：A
```

#### 4.3.2 拼写题

**需求描述**：根据提示，拖拽字母拼出单词。

**业务规则**：
- 显示单词的中文释义和首字母提示
- 提供打乱的字母供拖拽
- 答对：对怪物造成伤害
- 答错：显示正确拼写

**示例**：
```
提示：玛蒂尔达非常聪明，她喜欢r_____（阅读）书籍
字母池：e, r, a, d
正确答案：read
```

#### 4.3.3 连线题

**需求描述**：将英文单词与中文释义连线。

**业务规则**：
- 左侧 4 个英文单词，右侧 4 个中文释义
- 玩家通过连线匹配
- 全部正确：对怪物造成大量伤害
- 有错误：显示正确答案，玩家掉血

#### 4.3.4 跟读题

**需求描述**：AI 朗读句子，玩家跟读，系统评分。

**业务规则**：
- 播放标准发音
- 玩家点击录音按钮跟读
- 系统通过语音识别评分（0-100 分）
- 分数 ≥ 80：完美攻击
- 分数 60-79：普通攻击
- 分数 < 60：攻击失败

**示例**：
```
句子：Even if you're little, you can do a lot.
评分标准：发音准确度、流利度、完整度
```
